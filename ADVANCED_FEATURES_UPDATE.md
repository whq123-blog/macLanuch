## 技术亮点

### 1. SwiftUI 拖拽系统
- 使用 `onDrag()` 和 `onDrop()` API
- 自定义 `DropDelegate` 处理拖放逻辑
- 支持拖到应用上创建文件夹
- 支持拖到空位重新排列

### 2. 动画系统
- 使用 SwiftUI `Animation` 和 `withAnimation`
- 抖动动画：`repeatForever(autoreverses: true)`
- 自定义 `ViewModifier` 封装动画

### 3. 数据持久化
- 使用 `UserDefaults` 存储 JSON 数据
- `Codable` 协议序列化/反序列化
- 自动保存和恢复布局

### 4. 状态管理
- `ObservableObject` 和 `@Published`
- Combine 框架响应式编程
- MVVM 架构清晰分层
